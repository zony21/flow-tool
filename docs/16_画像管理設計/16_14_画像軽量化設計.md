# 16_14_画像軽量化設計

## 1. 目的

Canvas表示と一覧表示を安定させるため、画像を用途ごとに扱いやすい形へ整える方針を定義する。

## 2. 基本方針

- 元画像の管理情報は保持する。
- 一覧やCanvasでは軽量版を利用できる。
- 生成失敗時も元画像は維持する。

## 3. 対象

- 一覧表示用画像
- Canvas表示用画像
- Preview表示用画像

## 4. 処理タイミング

- Upload完了後
- 再生成要求時
- 表示設定変更時

## 5. テスト観点

- 元画像を保持できる。
- 軽量版を生成できる。
- 生成失敗時も登録済み画像を参照できる。

## 6. 完了条件

表示用途別の画像利用方針が定義されている。
