# 画像管理設計

## 目的
ノード・レーン・テンプレートで利用する画像・アイコンを一元管理する。

## 管理対象
- ノードアイコン
- レーンアイコン
- コメント添付画像
- テンプレート画像
- サムネイル

## 保存方式
- SQLiteにはメタデータのみ保持
- ファイルはストレージ保存
- 将来クラウドストレージ対応

## メタデータ
- ImageId
- FileName
- MimeType
- Width
- Height
- Size
- Hash
- CreatedAt
- UpdatedAt

## 機能
- アップロード
- ダウンロード
- 削除
- リネーム
- プレビュー
- サムネイル生成

## バリデーション
- MIMEチェック
- サイズ上限
- 拡張子チェック
- 重複(Hash)判定

## 将来拡張
- SVG最適化
- CDN配信
- バージョン管理
- AI画像解析
