# 16_01_画像管理概要

## 1. 目的

本書はAI Flow Designerの画像管理機能全体方針を定義する。

画像は単なる装飾ではなく、設備写真、画面仕様、構成図などAIがFlowを理解するための設計情報として扱う。

## 2. 基本方針

- 画像本体とMetadataを分離する。
- DBには画像バイナリを保存しない。
- Storage参照方式を採用する。
- 初期はローカルStorageを利用する。
- 将来クラウドStorageへ移行可能にする。
- Image Nodeと連携する。
- Version Snapshot、Exportとの整合性を保証する。

## 3. 管理対象

- PNG
- JPG / JPEG
- SVG
- WebP

## 4. 利用箇所

- Image Node
- Comment添付
- 設計資料
- AIレビュー
- Export

## 5. 対象外

- Undo / Redo履歴
- Canvas倍率
- Selection状態

## 6. 完了条件

画像管理の責務、保存方式、利用範囲が明確である。
