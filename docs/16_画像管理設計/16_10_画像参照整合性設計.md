# 16_10_画像参照整合性設計

## 1. 目的

画像データと利用箇所の整合性管理方針を定義する。

## 2. 基本方針

- 利用中画像を保護する。
- 削除前に利用状況を確認する。
- Flow、Template、Versionで参照状態を維持する。

## 3. 確認対象

- Image Node
- Comment添付
- Template
- Version Snapshot

## 4. Validation

確認内容:

- 画像情報が存在すること
- 利用権限があること
- Storage上に存在すること

## 5. 完了条件

画像と利用データの関係が安全に管理されている。
