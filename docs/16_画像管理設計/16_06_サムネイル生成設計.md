# 16_06_サムネイル生成設計

## 1. 目的

画像一覧、Preview、Canvas表示を高速化するためのサムネイル生成仕様を定義する。

## 2. 基本方針

- 元画像とは別にサムネイルを保持する。
- 自動生成する。
- 再生成可能なデータとして扱う。
- DBにはサムネイル実体を保存しない。

## 3. 生成タイミング

- 画像Upload完了時
- サイズ変更設定変更時
- 再生成要求時

## 4. 保存情報

保持項目:

- thumbnailPath
- width
- height
- generatedAt

## 5. 利用箇所

- Image一覧
- Image Node縮小表示
- Preview

## 6. 障害時

生成失敗しても元画像登録は可能とする。
後続で再生成する。

## 7. 完了条件

サムネイル生成、保存、利用方針が定義されている。
