# 16_05_画像Node連携設計

## 1. 目的

Image Nodeと画像管理機能の連携仕様を定義する。

## 2. 基本方針

- Nodeは画像本体を保持しない。
- IMAGE_IDのみ参照する。
- 表示時にImage Serviceから取得する。
- AI解析時には画像Metadataも利用する。

## 3. Image Node項目

- nodeId
- imageId
- caption
- description
- aiNote

## 4. 更新

画像差替時もNode IDは維持する。

## 5. Version連携

Snapshotでは参照画像IDを保持する。

## 6. 完了条件

Image Nodeと画像管理の責務が分離されている。
