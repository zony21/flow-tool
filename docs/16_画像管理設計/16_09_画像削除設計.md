# 16_09_画像削除設計

## 1. 目的

画像削除処理の仕様を定義する。

## 2. 基本方針

- 論理削除を基本とする。
- 参照中画像は即物理削除しない。
- 未使用画像のみ削除対象にする。

## 3. 削除前確認

確認対象:

- Image Node参照
- Comment参照
- Version Snapshot参照

## 4. 削除方式

通常削除:

- IS_DELETED更新

Cleanup:

- Storage物理削除

## 5. テスト観点

- 使用中画像を保護できる。
- 未使用画像を削除できる。

## 6. 完了条件

画像削除と参照保護方針が定義されている。
