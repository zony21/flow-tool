# 22_02_AIレビュー入出力仕様

## 1. 目的

AI Review Engineの入力・出力仕様を定義する。

## 2. Input Schema

```json
{
  "projectId": "string",
  "flowId": "string",
  "aiDslVersion": "v1",
  "structuredData": {},
  "aiDsl": "string",
  "settings": {
    "reviewStrictness": "normal",
    "detectMissingProcess": true,
    "detectResponsibilityIssue": true,
    "detectDbImpact": true,
    "detectApiImpact": true
  },
  "rules": []
}
```

## 3. Output Schema

```json
{
  "summary": "string",
  "results": [
    {
      "level": "warning",
      "category": "responsibility",
      "code": "NODE_WITHOUT_OWNER",
      "message": "担当が不明なNodeがあります。",
      "targetType": "node",
      "targetId": "node-001",
      "confidence": 0.82,
      "recommendation": "Laneまたは責務を設定してください。"
    }
  ]
}
```

## 4. level

| level | 内容 |
| --- | --- |
| error | 実装・設計に重大な影響がある |
| warning | 修正推奨 |
| suggestion | 改善提案 |

## 5. category

| category | 内容 |
| --- | --- |
| process | 処理抜け |
| condition | 条件分岐 |
| responsibility | 責務 |
| db | DB影響 |
| api | API影響 |
| communication | 外部通信 |
| error_handling | Error処理 |
| test | Test観点 |

## 6. 完了条件

- 入力Schemaが定義されている
- 出力Schemaが定義されている
- level/categoryが定義されている
