# 22_01_AIレビュー概要

## 1. 目的

AI Reviewの全体方針を定義する。

AI Reviewは、AI Flow Designerで作成されたFlowをAIが解析し、設計上の不備候補を提示する機能である。

## 2. Review対象

- 処理抜け
- 条件分岐の不明確さ
- Loop終了条件不足
- 責務違反
- DB影響不足
- API影響不足
- 外部通信漏れ
- Error処理不足
- Test観点不足

## 3. 入力

AI Reviewの入力は以下とする。

- Flow構造化データ
- AI DSL
- Project Setting
- AI Setting
- Export Setting
- Review Rule

## 4. 出力

AI Review結果は以下の形式で扱う。

```text
ReviewResult {
  level: error | warning | suggestion
  category
  message
  targetType
  targetId
  confidence
  recommendation
}
```

## 5. confidence

confidenceは0.0〜1.0で表す。

| 値 | 意味 |
| --- | --- |
| 0.8以上 | 高確度 |
| 0.5〜0.79 | 中確度 |
| 0.5未満 | 参考程度 |

## 6. 完了条件

- AI Reviewの対象が定義されている
- 入力と出力が定義されている
- confidenceの扱いが定義されている
