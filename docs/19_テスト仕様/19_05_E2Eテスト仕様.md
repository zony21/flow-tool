# 19_05_E2Eテスト仕様

## 1. 目的

ユーザー操作の開始から保存・出力・権限制御までを通しで確認するE2Eテスト仕様を定義する。

E2E Testでは、Frontend、Backend、DB、Exportが連携して正しく動作することを確認する。

## 2. 基本方針

- 実ユーザーに近い操作手順で確認する
- 画面表示だけでなくDB保存結果も確認する
- Role別に操作結果を確認する
- Export出力まで確認する
- 重要な業務導線を優先する

## 3. 主要シナリオ

| No | シナリオ | 優先度 |
| --- | --- | --- |
| E2E-001 | Project作成 | A |
| E2E-002 | Flow作成 | A |
| E2E-003 | Node / Link編集 | A |
| E2E-004 | 保存・再読込 | A |
| E2E-005 | Version作成 | A |
| E2E-006 | Viewer読み取り専用 | A |
| E2E-007 | Export実行 | A |
| E2E-008 | 設定変更 | B |
| E2E-009 | Template適用 | B |
| E2E-010 | Image Node利用 | B |

## 4. E2E-001 Project作成

手順:

1. ログインする
2. Project一覧を開く
3. 新規Project作成を押す
4. Project名を入力する
5. 保存する

期待結果:

- Projectが作成される
- 作成者がOwnerになる
- Project Setting / AI Setting / Export SettingのDefaultが利用可能になる

## 5. E2E-002 Flow作成

手順:

1. Projectを開く
2. 新規Flowを作成する
3. Flow名を入力する
4. Flow Editorを開く

期待結果:

- Flowが作成される
- Editor画面が表示される
- 初期Canvasが表示される

## 6. E2E-003 Node / Link編集

手順:

1. Node PaletteからStartを追加する
2. Processを追加する
3. Endを追加する
4. Start -> Process -> Endを接続する
5. ProcessのLabelを変更する

期待結果:

- Nodeが3件作成される
- Linkが2件作成される
- Property Panelの変更がStoreに反映される

## 7. E2E-004 保存・再読込

手順:

1. Flowを保存する
2. 画面を再読み込みする
3. 同じFlowを開く

期待結果:

- 保存前と同じNode / Linkが表示される
- DBの構造化データと画面表示が一致する

## 8. E2E-005 Version作成

手順:

1. Flow編集後にVersion作成を押す
2. Version名を入力する
3. Version一覧を開く

期待結果:

- Snapshotが作成される
- Version一覧に表示される
- Snapshot内容が作成時点のFlowと一致する

## 9. E2E-006 Viewer読み取り専用

手順:

1. Viewerでログインする
2. Projectを開く
3. Flowを開く
4. Node移動、Node追加、保存を試す

期待結果:

- Flow表示は可能
- Node移動不可
- Node追加不可
- 保存不可
- Exportは可能

## 10. E2E-007 Export実行

手順:

1. Flowを開く
2. Export Dialogを開く
3. Mermaidを出力する
4. JSONを出力する
5. AI DSLを出力する

期待結果:

- MermaidにNode / Linkが含まれる
- JSONにProject / Flow / Node / Linkが含まれる
- AI DSLにVersion / Node / Link / Metadataが含まれる

## 11. E2E-008 設定変更

手順:

1. OwnerでProject設定を開く
2. Export設定を変更する
3. 保存する
4. Export Dialogを開く

期待結果:

- 設定変更が保存される
- Export Dialogに設定が反映される

## 12. 完了条件

- 主要導線が通しで確認できる
- 保存・再読込後も構造化データが維持される
- Viewer読み取り専用が確認できる
- Export出力が確認できる
- 設定変更が出力動作へ反映される
