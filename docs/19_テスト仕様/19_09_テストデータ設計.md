# 19_09_テストデータ設計

## 1. 目的

AI Flow Designerのテストで使用する標準テストデータを定義する。

テストデータを固定することで、Unit / API / Frontend / E2E / Export / AI DSLの期待結果を安定させる。

## 2. 基本方針

- 最小構成から複雑構成まで段階的に用意する
- Role別Userを必ず用意する
- Decision、Loop、Image、Comment、Exportを確認できるFlowを用意する
- AI DSL検証用に責務・通信・DB/API情報を含むデータを用意する

## 3. Userデータ

| User | Role | 説明 |
| --- | --- | --- |
| ownerUser | Owner | Project所有者 |
| editorUser | Editor | Flow編集者 |
| viewerUser | Viewer | 閲覧者 |
| outsideUser | なし | Project未参加 |
| suspendedUser | suspended | 利用停止Member |

## 4. Projectデータ

| Project | 目的 |
| --- | --- |
| EmptyProject | 新規Project、Default設定確認 |
| BasicFlowProject | 基本Flow確認 |
| DecisionFlowProject | 条件分岐確認 |
| LoopFlowProject | Loop許可確認 |
| ImageFlowProject | Image Node確認 |
| CommentFlowProject | Comment確認 |
| PermissionProject | Role別権限確認 |
| ExportProject | Export確認 |
| AiDslProject | AI DSL確認 |

## 5. BasicFlow

構成:

```text
Start -> Process -> End
```

Node:

| Node | Type | Label |
| --- | --- | --- |
| start-001 | Start | 開始 |
| process-001 | Process | 処理A |
| end-001 | End | 終了 |

Link:

| Source | Target |
| --- | --- |
| start-001 | process-001 |
| process-001 | end-001 |

## 6. DecisionFlow

構成:

```text
Start -> Decision
Decision -- Yes --> ProcessA -> End
Decision -- No --> ProcessB -> End
```

確認内容:

- Decision Node
- 条件分岐
- Yes/No Link
- Mermaid分岐
- AI DSL condition

## 7. LoopFlow

構成:

```text
Start -> ProcessA -> Decision
Decision -- retry --> ProcessA
Decision -- complete --> End
```

確認内容:

- Loop Link許可
- 無限展開しないExport
- AI DSL loop表現

## 8. ImageFlow

確認内容:

- Image Node
- Image Metadata
- PDF出力時の画像含有
- AI DSL image情報

## 9. CommentFlow

確認内容:

- Node紐付Comment
- 独立Comment
- PDF / JSON / AI DSLへの出力

## 10. AiDslProject

AI DSL確認用に以下を含む。

- Lane
- Stage
- Decision
- Loop
- DB操作Node
- API通信Node
- PLC通信Node
- Comment
- Image
- Metadata

## 11. Settingデータ

| 設定 | パターン |
| --- | --- |
| User Setting | Default / dark / compact |
| Project Setting | Viewer Export許可 / 禁止 |
| Editor Setting | Grid ON / OFF、Snap ON / OFF |
| AI Setting | simple / standard / detailed |
| Export Setting | A4 / A3、LR / TB、metadata有無 |

## 12. 完了条件

- Role別Userが定義されている
- 主要Flowパターンが定義されている
- ExportとAI DSLに必要なテストデータが定義されている
- 20_実装タスクで初期Seed作成タスクに展開できる
