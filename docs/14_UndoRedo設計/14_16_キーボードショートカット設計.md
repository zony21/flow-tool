# 14_16_キーボードショートカット設計

## 1. 目的

本書は、AI Flow DesignerのUndo / Redoに関するキーボードショートカット設計を定義する。

Undo / Redoは頻繁に利用されるため、Toolbar操作だけでなくキーボード操作に対応する。

## 2. 基本方針

- Ctrl + ZでUndoする。
- Ctrl + Shift + ZでRedoする。
- 将来、Ctrl + YもRedoとして対応する。
- Input / Textarea入力中はブラウザ標準編集を優先する。
- Dialog表示中はDialogの操作を優先する。

## 3. Shortcut一覧

| Shortcut | 操作 |
| --- | --- |
| Ctrl + Z | Undo |
| Ctrl + Shift + Z | Redo |
| Ctrl + Y | Redo 将来対応 |

## 4. Event Guard

以下ではUndo / Redoを実行しない。

- Textarea入力中
- Input入力中
- ContentEditable入力中
- Dialog表示中
- IME変換中
- 保存中
- Flow未読込

## 5. Toolbar連携

ToolbarのUndo / RedoボタンはStack状態により活性制御する。

- undoStackが空ならUndo非活性
- redoStackが空ならRedo非活性

## 6. 禁止事項

- 入力欄編集中にFlow全体Undoを実行する。
- Dialog表示中に背面CanvasのUndoを実行する。
- Redo Stackが空でもRedoを実行する。

## 7. テスト観点

- Ctrl + ZでUndoできる。
- Ctrl + Shift + ZでRedoできる。
- Input入力中はFlow Undoが実行されない。
- Dialog表示中はFlow Undoが実行されない。

## 8. 完了条件

- Undo / RedoショートカットとGuard条件が定義されている。
- Toolbarとの活性制御が定義されている。
