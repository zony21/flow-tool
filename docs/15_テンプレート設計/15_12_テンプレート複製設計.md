# 15_12_テンプレート複製設計

## 1. 目的

本書は、AI Flow DesignerのTemplate複製設計を定義する。

Template複製は、既存Templateを元にProject用Templateを作成するための機能である。

## 2. 複製対象

- Standard Template
- Project Template
- Imported Template

## 3. 基本方針

- 複製元TemplateJsonをコピーする。
- 新しいtemplateIdを発行する。
- templateNameは重複しないよう補正する。
- 複製先Projectを指定する。
- 複製元は変更しない。

## 4. 複製フロー

```text
Template選択
  ↓
複製先Project確認
  ↓
TemplateJsonコピー
  ↓
新Template作成
  ↓
Project Templateとして保存
```

## 5. 名前補正

同名Templateが存在する場合、末尾に連番を付与する。

例:

```text
基本フロー コピー
基本フロー コピー 2
```

## 6. 権限

複製元の参照権限と、複製先Projectの作成権限が必要。

## 7. テスト観点

- Standard TemplateをProject Templateへ複製できる。
- Project Templateを同一Project内で複製できる。
- 同名の場合に名前補正される。
- 複製元が変更されない。

## 8. 完了条件

- Template複製対象、フロー、名前補正、権限が定義されている。
