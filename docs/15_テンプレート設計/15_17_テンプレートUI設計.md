# 15_17_テンプレートUI設計

## 1. 目的

本書は、AI Flow DesignerのTemplate UI設計を定義する。

Template UIは、Templateの検索、Preview、保存、適用、複製、Import / Exportを行うための画面・Dialogである。

## 2. 基本方針

- Template一覧はDialogまたは専用Panelで表示する。
- 一覧では軽量情報のみ表示する。
- Previewで適用前に内容を確認できる。
- Append / Replace / New Flowを選択できる。
- Replaceは確認Dialogを必須にする。

## 3. UI構成

```text
TemplateDialog
  ├─ SearchArea
  ├─ CategoryFilter
  ├─ TemplateList
  ├─ TemplatePreview
  └─ ActionArea
```

## 4. Template保存Dialog

入力項目:

- templateName
- description
- category
- tags
- saveRange

saveRange:

- Flow全体
- 選択範囲

## 5. Template適用Dialog

表示項目:

- Template名
- 説明
- Lane数
- Stage数
- Node数
- Link数
- Comment数

操作:

- Append
- Replace
- New Flow
- Cancel

## 6. Template Preview

初期は数値サマリと代表Node一覧を表示する。
将来Canvas Previewへ拡張する。

## 7. Error表示

- Template読込失敗
- 権限不足
- SchemaVersion非対応
- Validation Error

## 8. テスト観点

- Template一覧を表示できる。
- 検索できる。
- Previewを表示できる。
- Replace時に確認Dialogが出る。
- 保存DialogでTemplateを作成できる。

## 9. 完了条件

- Template一覧、保存、適用、Preview UIが定義されている。
