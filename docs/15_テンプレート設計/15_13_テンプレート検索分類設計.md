# 15_13_テンプレート検索分類設計

## 1. 目的

本書は、AI Flow DesignerのTemplate検索・分類設計を定義する。

Template数が増えた場合でも、用途に合うTemplateを素早く探せるようにする。

## 2. 基本方針

- Template名で検索できる。
- Categoryで分類できる。
- Tagsで絞り込める。
- Standard / Projectを切り替えられる。
- 一覧ではTemplateJson全体を取得しない。

## 3. 検索条件

- keyword
- templateType
- category
- tags
- projectId
- updatedAt

## 4. Category候補

- 基本
- 業務フロー
- API連携
- DB更新
- PLC連携
- 画像あり
- AIレビュー向け

## 5. Tags

Tagsは複数指定可能とする。

例:

- WCS
- AGF
- PLC
- API
- DB
- 承認
- 分岐

## 6. 一覧性能

Template一覧では以下のみ取得する。

- templateId
- templateName
- description
- category
- tags
- templateType
- updatedAt
- nodeCount

## 7. テスト観点

- keywordで検索できる。
- categoryで絞り込める。
- tagで絞り込める。
- 一覧でTemplateJsonを返さない。

## 8. 完了条件

- 検索条件、分類、タグ、一覧性能方針が定義されている。
