# 15_18_エラー競合設計

## 1. 目的

本書は、AI Flow DesignerのTemplate機能におけるエラー・競合設計を定義する。

TemplateはFlow構造へ影響するため、読込、保存、適用、Importでの不整合を安全に扱う必要がある。

## 2. エラー分類

- Template Not Found
- Permission Denied
- Duplicate Name
- Invalid Template Json
- Unsupported Schema Version
- Invalid Node Type
- Broken Link Reference
- Broken Comment Reference
- Import File Invalid

## 3. 保存時エラー

- Template名未入力
- Template名重複
- 保存対象なし
- 選択範囲内の参照不整合

## 4. 適用時エラー

- Templateが削除済み
- SchemaVersion非対応
- NodeType未対応
- Link参照不整合
- Comment参照不整合
- Replace確認なし

## 5. Import時エラー

- JSON形式不正
- サイズ超過
- SchemaVersion非対応
- 必須項目不足

## 6. 競合

Template更新時はupdatedAtまたはrevisionを確認する。

競合時は上書きせず、再読込を促す。

## 7. Error表示

FrontendではDialog内にErrorを表示する。
詳細はtraceId付きで表示し、再試行導線を出す。

## 8. テスト観点

- Template名重複を検出できる。
- 削除済Template適用を拒否できる。
- 非対応SchemaVersionを拒否できる。
- Import不正JSONを拒否できる。

## 9. 完了条件

- Template機能の主要エラーと競合対応が定義されている。
