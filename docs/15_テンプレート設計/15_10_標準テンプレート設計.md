# 15_10_標準テンプレート設計

## 1. 目的

本書は、AI Flow Designerの標準Template設計を定義する。

標準Templateは、システムが初期提供する汎用Templateであり、ユーザーが新規Flow作成時に利用できる。

## 2. 基本方針

- 標準Templateは全Projectから参照可能とする。
- 初期提供数は少数にする。
- 業務特化しすぎない。
- 更新時はSchemaVersionを意識する。
- ユーザーは標準Templateを直接編集できない。
- 編集したい場合はProject Templateへ複製する。

## 3. 初期候補

- 基本フロー
- Laneあり業務フロー
- 判定分岐ありフロー
- API連携フロー
- DB更新ありフロー
- PLC / 設備連携フロー

## 4. 管理項目

- templateName
- description
- category
- tags
- templateJson
- schemaVersion
- isStandard

## 5. 利用フロー

```text
Template一覧
  ↓
Standard Template選択
  ↓
Preview
  ↓
Append / Replace / New Flow
```

## 6. 編集制御

標準Templateは読み取り専用とする。
Project内で変更したい場合は複製してProject Templateとして保存する。

## 7. テスト観点

- 標準Templateを一覧表示できる。
- 標準Templateを適用できる。
- 標準Templateを直接編集できない。
- Project Templateへ複製できる。

## 8. 完了条件

- 標準Templateの目的、初期候補、編集制御が定義されている。
