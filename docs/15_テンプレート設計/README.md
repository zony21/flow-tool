# 15_テンプレート設計

## 1. 本書の位置付け

本フォルダは、AI Flow Designerのテンプレート機能設計を管理する。

Templateは、Lane / Stage / Node / Link / Commentを再利用可能な構造として保存・適用するための機能である。単なる見た目の複製ではなく、SSOTの一部構造を安全に再利用する仕組みである。

## 2. 設計思想

- Templateは構造化データとして保持する。
- 適用時はIDを必ず再採番する。
- Lane / Stage / Node / Link / Commentを含められる。
- AI専用メモも保持できる。
- 標準TemplateとProject Templateを分ける。
- Template適用はUndo / Redo可能な複合Commandとして扱う。
- Version Snapshotとは責務を分離する。
