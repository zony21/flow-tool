# 02_05_機能一覧

## 1. 本書の目的

本書は、AI Flow Designer の機能一覧を定義する。

機能一覧は、画面設計、API設計、DB設計、フロントエンド実装、バックエンド実装、テスト仕様、実装タスクの対応表として利用する。

## 2. 機能分類

機能は以下に分類する。

- 認証
- Project管理
- Flow管理
- Flow編集
- Undo / Redo
- AutoSave
- Version管理
- Template管理
- Export
- 画像管理
- 権限管理
- 設定

- 権限判定
