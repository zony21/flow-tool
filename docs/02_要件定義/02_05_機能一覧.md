# 02_05_機能一覧

## 1. 本書の目的

本書は、AI Flow Designer の機能一覧を定義する。

機能一覧は、画面設計、API設計、DB設計、フロントエンド設計、バックエンド設計、テスト仕様、実装タスクを結び付ける索引として利用する。

## 2. 機能分類

|分類|概要|
|---|---|
|認証|GitHub OAuth、JWT、ログアウト|
|Project|Project作成、参照、更新、削除|
|Flow|Flow作成、参照、更新、削除、複写|
|Editor|Lane、Stage、Node、Link、Comment編集|
|UndoRedo|Command履歴管理|
|AutoSave|差分保存|
|Version|Snapshot保存、比較、復元|
|Template|テンプレート作成、適用、検索|
|Export|Mermaid、PDF、JSON等の出力|
|Image|画像アップロード、参照、削除|
|Permission|メンバー、権限、編集ロック|
|Setting|ユーザー設定、Project設定|

## 4. Editor系機能詳細

Editor系機能は、画面操作としては個別だが、保存はFlow保存またはAutoSaveでまとめて行う。

対象:

- Lane
- Stage
- Node
- Link
- Comment

これらはCommand化され、Undo / Redo対象となる。

## 5. AutoSave対象

AutoSave対象機能:

- Node編集
- Link編集
- Lane編集
- Stage編集
- Comment編集
- Template適用
- Flowプロパティ変更

AutoSave対象外:

- 選択状態
- Hover状態
- Zoom / Pan
- Tooltip表示

## 6. Version対象

Version Snapshotに含める対象:

- Flow
- Lane
- Stage
- Node
- Link
- Comment
- Layout

含めない対象:

- Undo / Redo履歴
- 選択状態
- 編集ロック
- AutoSave一時状態

## 7. 監査ログ対象

監査対象候補:

- Project作成 / 更新 / 削除
- Flow作成 / 削除
- Version復元
- メンバー追加 / 削除
- 権限変更
- Template削除

初期実装では監査カラムを必須とし、監査ログ専用画面は将来対応とする。
