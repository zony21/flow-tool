# 18_04_API設計

## 1. 目的

AI Flow Designerの設定機能に関するAPI仕様を定義する。

設定APIは、User個人設定、Project共通設定、Editor設定、AI設定、Export設定の取得・更新を行う。

## 2. 基本方針

- 設定分類ごとにAPIを分ける
- GET時に設定レコードが存在しない場合はDefault値を返す
- 更新時はBackendでValidationする
- Project単位設定は17章のPermissionで認可する
- User個人設定は本人のみ更新可能とする

## 3. API一覧

| API | Method | 内容 | Permission |
| --- | --- | --- | --- |
| /api/settings/user | GET | 自分のUser Setting取得 | ログイン済み |
| /api/settings/user | PUT | 自分のUser Setting更新 | Setting.UserUpdate |
| /api/projects/{projectId}/settings/project | GET | Project Setting取得 | Project.Read |
| /api/projects/{projectId}/settings/project | PUT | Project Setting更新 | Setting.ProjectUpdate |
| /api/projects/{projectId}/settings/editor | GET | Editor Setting取得 | Flow.Read |
| /api/projects/{projectId}/settings/editor | PUT | Editor Setting更新 | Setting.UserUpdate |
| /api/projects/{projectId}/settings/ai | GET | AI Setting取得 | Project.Read |
| /api/projects/{projectId}/settings/ai | PUT | AI Setting更新 | Setting.AiUpdate |
| /api/projects/{projectId}/settings/export | GET | Export Setting取得 | Project.Read |
| /api/projects/{projectId}/settings/export | PUT | Export Setting更新 | Setting.ExportUpdate |

## 4. User Setting API

### 4.1 GET /api/settings/user

ログインUserのUser Settingを取得する。

Response:

```json
{
  "theme": "system",
  "language": "ja",
  "defaultZoom": 1.0,
  "density": "normal",
  "showMiniMap": true,
  "showGrid": true,
  "autoSave": false
}
```

### 4.2 PUT /api/settings/user

ログインUserのUser Settingを更新する。

Request:

```json
{
  "theme": "dark",
  "language": "ja",
  "defaultZoom": 1.0,
  "density": "normal",
  "showMiniMap": true,
  "showGrid": true,
  "autoSave": false
}
```

Validation:

- theme: light / dark / system
- language: ja / en
- defaultZoom: 0.2〜3.0
- density: compact / normal / comfortable

## 5. Project Setting API

### 5.1 GET /api/projects/{projectId}/settings/project

Project共通設定を取得する。

Response:

```json
{
  "projectNameRule": "free",
  "flowNameRule": "free",
  "defaultExportFormat": "pdf",
  "allowViewerExport": true,
  "versionRequiredOnExport": false,
  "archiveLocked": true
}
```

### 5.2 PUT /api/projects/{projectId}/settings/project

Project共通設定を更新する。

Permission:

```text
Setting.ProjectUpdate
```

## 6. Editor Setting API

### 6.1 GET /api/projects/{projectId}/settings/editor

ログインUserの対象ProjectにおけるEditor Settingを取得する。

Response:

```json
{
  "gridEnabled": true,
  "snapEnabled": true,
  "gridSize": 16,
  "defaultLinkType": "bezier",
  "showLaneHeader": true,
  "showStageHeader": true,
  "nodeLabelVisible": true,
  "canvasBackground": "default"
}
```

### 6.2 PUT /api/projects/{projectId}/settings/editor

ログインUserのEditor Settingを更新する。

Permission:

```text
Setting.UserUpdate
```

Editor Settingは個人設定扱いのため、Ownerでなくても更新可能とする。

## 9. エラー仕様

| 状態 | HTTP Status | レスポンス例 |
| --- | --- | --- |
| 未ログイン | 401 | AUTH_REQUIRED |
| 権限不足 | 403 | PERMISSION_DENIED |
| Projectなし | 404 | PROJECT_NOT_FOUND |
| Validation Error | 400 | VALIDATION_ERROR |

共通Error Response:

```json
{
  "errorCode": "VALIDATION_ERROR",
  "message": "設定値が不正です。",
  "details": [
    {
      "field": "theme",
      "message": "themeはlight/dark/systemのいずれかです。"
    }
  ]
}
```

## 10. 完了条件

- 設定分類ごとの取得・更新APIが定義されている
- Permissionが定義されている
- Request / Responseの形が明確である
- ValidationとErrorが定義されている
