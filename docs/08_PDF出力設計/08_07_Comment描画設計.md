# 08_07_Comment描画設計

## 1. 目的
CommentをPDFへ出力し、設計意図、補足、AI専用メモをレビュー可能にする。

## 2. Comment種別
| 種別 | 出力方針 |
| --- | --- |
| 独立コメント | Canvas上の位置へ吹き出し表示する |
| Nodeコメント | 対象Node近辺へ番号を表示し、一覧に全文を出す |
| AI専用メモ | 図中には任意表示、一覧へ出力する |

## 3. 描画方針
図の可読性を優先し、長文Commentは図中に全文表示しない。図中は番号または短縮文とし、一覧ページで全文を確認する。

## 4. 出力オプション
- コメントを図中に表示する
- コメント一覧を出力する
- AI専用メモを含める

## 5. 完了条件
Commentが図と一覧のどちらかで必ず確認でき、Node紐付けが失われない。
