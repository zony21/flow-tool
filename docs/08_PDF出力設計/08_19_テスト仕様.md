# 08_19_テスト仕様

## 1. 目的
PDF出力機能がSSOTから正しくPDFを生成できることを確認する。

## 2. 単体テスト
- ExportOptions validation
- FlowSnapshot to PdfViewModel変換
- Canvas座標からPDF座標への変換
- 縮尺計算
- ページ分割計算
- Node / Link / Comment件数一致

## 3. 結合テスト
- PDF Export API成功
- 権限なし403
- 存在しないVersionで404
- 不正Optionsで400
- 画像取得不可時のエラー表示

## 4. 表示確認
- 日本語文字化けなし
- Node形状識別可能
- Link矢印とラベル表示
- Comment一覧出力
- ページ番号表示

## 5. 受入条件
保存済みFlowVersionからPDFを出力し、図・一覧・コメント・メタ情報が欠落なく確認できる。
