# 08_10_フォント設計

## 1. 目的
日本語を含むPDFを文字化けなく出力し、Windows、macOS、ブラウザ、印刷環境で安定して閲覧できるようにする。

## 2. 方針
- 日本語フォントを明示的に指定する。
- PDF生成時に必要なフォントを埋め込む。
- フォントファイルはリポジトリへ同梱しない。
- 実行環境の設定または配置先パスで管理する。

## 3. 初期フォント
初期候補はNoto Sans CJK JP相当とする。ただし実体ファイルは環境設定で参照する。

## 4. 代替処理
指定フォントが存在しない場合はPDF生成を警告付きで停止する。文字化けしたPDFを正常出力として返さない。

## 5. テスト観点
- 日本語
- 英数字
- 記号
- 長文
- 半角全角混在

## 6. 完了条件
Node名、Comment、一覧表、ヘッダー、フッターの全てで日本語が正しく表示される。
