# 08_05_図形描画設計

## 1. 目的
Node、Lane、StageをPDF上へ再現し、人が見て工程・責務・判断を理解できる図として出力する。

## 2. Node描画
| Node種別 | PDF表現 |
| --- | --- |
| start | 角丸矩形または開始記号 |
| end | 角丸矩形または終了記号 |
| process | 矩形 |
| decision | ひし形 |
| hexagon | 六角形 |
| callout | 吹き出し |
| image | 画像矩形 |

## 3. Lane / Stage描画
LaneとStageは背景レイヤーとして描画し、Nodeより先に出力する。背景色は淡色化し、Nodeの可読性を優先する。

## 4. テキスト描画
Node名を主表示、説明を副表示とする。長文は折り返し、収まらない場合は一覧表で全文を確認できるようにする。

## 5. 完了条件
- Node種別ごとの形状が識別できる。
- Lane / Stageの境界が確認できる。
- Node名称が判読できる。
