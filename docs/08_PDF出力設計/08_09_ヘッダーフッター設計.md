# 08_09_ヘッダーフッター設計

## 1. 目的
PDFの各ページに識別情報を付与し、印刷・共有後もProject、Flow、Version、ページ位置を判別できるようにする。

## 2. ヘッダー表示項目
- Project名
- Flow名
- Version番号
- 機密区分

## 3. フッター表示項目
- ページ番号
- 総ページ数
- 出力日時
- 出力者

## 4. 表示ルール
表紙はヘッダーを表示しない。フロー図ページと一覧ページは同一フォーマットのヘッダー・フッターを表示する。

## 5. 完了条件
全ページでページ番号が連続し、PDF単体で対象FlowVersionを識別できる。
