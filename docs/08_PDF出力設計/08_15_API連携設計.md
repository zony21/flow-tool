# 08_15_API連携設計

## 1. 目的
FrontendからBackendへPDF出力要求を送信し、生成されたPDFを安全に受け取るAPI仕様を定義する。

## 2. API
`POST /api/projects/{projectId}/flows/{flowId}/versions/{versionId}/exports/pdf`

## 3. Request
```json
{
  "paperSize": "A4",
  "orientation": "landscape",
  "scaleMode": "fit",
  "includeComments": true,
  "includeAiMemo": true,
  "includeTables": true
}
```

## 4. Response
成功時は `application/pdf` を返す。ファイル名は `ProjectName_FlowName_vVersion.pdf` を基本とする。

## 5. 権限
Project閲覧権限とFlow閲覧権限が必要。非公開ProjectではMember以外の出力を禁止する。

## 6. 完了条件
FrontendからPDFをダウンロードでき、権限エラー、入力エラー、生成エラーを区別できる。
