# 08_06_Link描画設計

## 1. 目的
LinkをPDF上へ正確に描画し、処理順序、分岐条件、戻りループを読み取れるようにする。

## 2. 初期対応
初期はBezier Linkを正式対応とする。将来Straight、Step、SmoothStepを追加できるよう、LinkRendererは線種ごとに分離する。

## 3. 描画要素
- 線本体
- 矢印
- Linkラベル
- 条件式
- 接続元Node
- 接続先Node

## 4. ループ表現
Loopは許可する。自己参照Link、前工程へ戻るLink、分岐後の戻りLinkはエラーにしない。
PDFでは線の重なりを避けるため、ラベル位置と制御点を調整する。

## 5. 完了条件
- 全LinkがNode間に描画される。
- ラベルと条件が読める。
- 循環Linkが削除・省略されない。
