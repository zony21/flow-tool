# 08_PDF出力設計

## 1. 本書の位置付け

本フォルダは、AI Flow DesignerにおけるPDF出力機能の正式設計書を管理する。
PDFは画面キャプチャではなく、SSOTであるFlowVersion Snapshotから生成される正式成果物である。

## 3. 管理方針

`08_PDF出力設計` 配下には正式設計書のみを残す。
更新メモ、draft、temp、old、数行だけの旧設計書は正式設計書へ統合後に削除する。
