# 08_08_画像出力設計

## 1. 目的
画像Nodeおよび添付画像をPDFへ安全に出力し、設備写真、画面キャプチャ、補足図を設計書として共有できるようにする。

## 2. 対象画像
- 画像Nodeに設定された画像
- Commentに添付された画像
- 将来のテンプレート画像

## 3. 取得方針
画像は保存キーからBackendで取得する。Frontendから渡された一時URLを信用しない。

## 4. 描画方針
画像はNode矩形へ収まるようアスペクト比を維持して描画する。余白は中央寄せとし、画像が取得できない場合は代替枠を描画する。

## 5. セキュリティ
- 出力権限のない画像はPDFへ含めない。
- 外部URL画像は初期対応しない。
- MIMEと拡張子を検証する。

## 6. 完了条件
画像NodeがPDF上で欠落せず、取得不可時もエラー内容が確認できる。
