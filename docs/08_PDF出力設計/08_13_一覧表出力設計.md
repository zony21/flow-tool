# 08_13_一覧表出力設計

## 1. 目的
図だけでは読み切れない詳細情報を一覧表として出力し、設計書として利用できるPDFにする。

## 2. 出力一覧
- Node一覧
- Link一覧
- Lane一覧
- Stage一覧
- Comment一覧
- AI専用メモ一覧

## 3. Node一覧列
ID、種別、名称、説明、Lane、Stage、責務、座標、サイズ、AIメモ有無を表示する。

## 4. Link一覧列
ID、接続元、接続先、ラベル、条件、線種、説明を表示する。

## 5. Comment一覧列
ID、対象Node、コメント種別、本文、作成者、更新日時を表示する。

## 6. 完了条件
図中で省略された長文情報を一覧表で全文確認できる。
