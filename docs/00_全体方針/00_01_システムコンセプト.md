# 00_01_システムコンセプト

## 1. 本書の目的

本書は、AI Flow Designer のシステムコンセプトを定義する。

本システムは、単なる図作成ツールではなく、構造化されたFlowデータを正として保持し、そのデータから人間向けの図面とAI向けの出力を生成するための設計支援ツールである。

## 2. 中核思想

本システムの中核思想は Single Source of Truth である。

画面上の図形や線は表示結果であり、正となるデータはDBに保存される Project、Flow、Lane、Stage、Node、Link、Comment である。

この方針により、画面表示、JSON出力、Mermaid出力、PDF出力、Version復元、Template適用の結果を同じ情報源から安定して生成できる。

## 3. 完了条件

- 構造化Flowデータを正として保持できる
- 図面とExport結果が同じFlowデータから生成される
- Lane、Stage、Node、Link、Commentの意味情報が失われない
