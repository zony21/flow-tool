# 03_07_出力画面

## 1. 本書の目的

本書は、出力画面の詳細仕様を定義する。
出力画面は、FlowVersionに保存された構造化データから、Mermaid、PDF、JSONなどの成果物を生成するための画面である。

## 2. 画面の役割

本画面では以下を行う。

- 出力対象FlowVersionの確認
- 出力形式の選択
- 出力オプションの指定
- プレビュー表示
- ファイル生成
- 生成結果のダウンロード

## 3. 対象出力

初期対応は以下とする。

- Mermaid flowchart
- Mermaid sequenceDiagram
- PDF
- JSON

将来対応は以下とする。

- AI DSL
- API仕様
- DB更新一覧
- 通信一覧
- PLC一覧
- 設計書ドラフト
- コード生成用定義

## 4. レイアウト

```text
[ヘッダー]
[パンくず: プロジェクト > フロー > 出力]
[出力対象情報]
[出力形式選択]
[出力オプション]
[プレビュー]
[生成 / ダウンロード]
```

## 5. 表示項目

| 項目 | 内容 |
| --- | --- |
| プロジェクト名 | 対象Project |
| フロー名 | 対象Flow |
| バージョン | 対象FlowVersion |
| 最終更新日時 | 出力対象の更新日時 |
| ノード数 | 出力対象Node件数 |
| 接続数 | 出力対象Link件数 |

## 6. 操作仕様

### 6.1 出力形式選択

ユーザーは出力形式を選択する。
複数選択可能とするか単一選択とするかは初期実装で単一選択を基本とする。

### 6.2 プレビュー

MermaidとJSONは画面上でプレビューできる。
PDFは生成前の簡易プレビュー、または生成後プレビューとする。

### 6.3 生成

生成ボタン押下でExport APIを呼び出す。
生成中はボタンを非活性にし、二重実行を防止する。

### 6.4 ダウンロード

生成成功後、ファイルをダウンロードできる。
ファイル名にはプロジェクト名、フロー名、バージョン、出力日時を含める。

## 7. 出力オプション

| オプション | 内容 | 初期対応 |
| --- | --- | --- |
| コメント含有 | コメントを出力に含める | 対応 |
| AIメモ含有 | AI向けメモを含める | 将来 |
| レーン情報含有 | Lane情報を含める | 対応 |
| 工程情報含有 | Stage情報を含める | 対応 |
| 表示順優先 | 表示順に並べる | 対応 |

## 8. API連携

- POST /api/flow-versions/{flowVersionId}/export/mermaid
- POST /api/flow-versions/{flowVersionId}/export/pdf
- POST /api/flow-versions/{flowVersionId}/export/json

## 9. DB連携

- FLOW_VERSION
- LANE
- STAGE
- NODE
- LINK
- COMMENT

## 10. エラー表示

- 出力対象の取得に失敗しました
- Mermaid生成に失敗しました
- PDF生成に失敗しました
- JSON生成に失敗しました
- ダウンロードに失敗しました

## 11. テスト観点

- 出力対象情報が表示されること
- Mermaidを生成できること
- JSONを生成できること
- PDFを生成できること
- 生成中に二重実行できないこと
- エラー時にメッセージが表示されること

## 12. 完了条件

本画面は、FlowVersionから初期対応成果物を生成し、利用者が確認・取得できる仕様が定義された場合に完了とする。
