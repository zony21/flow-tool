# 03_08_テンプレート選択画面

## 1. 本書の目的

本書は、テンプレート選択画面の詳細仕様を定義する。
テンプレート選択画面は、利用者があらかじめ定義されたレーン、工程、ノードをもとにFlowを作成するための画面である。

## 2. 画面の役割

本画面では以下を行う。

- テンプレート一覧表示
- テンプレート詳細確認
- テンプレートからFlow作成
- 既存Flow複製との違いの明確化

## 3. テンプレートの考え方

テンプレートはレーン、工程、ノードまで含める。
単なる見た目のひな形ではなく、構造化データの初期セットとして扱う。

例として、包装ラインテンプレートでは以下を含める。

- 包装PLC
- 統括PLC
- WCS
- RCS
- RFID工程
- 搬送要求工程
- ラベル印刷工程
- 基本ノード

## 4. レイアウト

```text
[テンプレート検索]
[テンプレート一覧]
[テンプレート概要]
[含まれるレーン・工程・ノード]
[作成ボタン / キャンセル]
```

## 5. 表示項目

| 項目 | 内容 |
| --- | --- |
| テンプレート名 | テンプレート表示名 |
| カテゴリ | 業務分類 |
| 説明 | テンプレート説明 |
| レーン数 | 含まれるLane数 |
| 工程数 | 含まれるStage数 |
| ノード数 | 含まれるNode数 |
| 更新日時 | 最終更新日時 |

## 6. 操作仕様

### 6.1 テンプレート選択

一覧からテンプレートを選択すると、右側または下部に詳細を表示する。

### 6.2 テンプレート詳細確認

テンプレートに含まれるレーン、工程、ノードの概要を確認できる。

### 6.3 Flow作成

作成ボタン押下で、選択テンプレートをもとにFlowを作成する。
作成時にはFlow名を入力させる。

## 7. バリデーション

| 項目 | 条件 |
| --- | --- |
| Flow名 | 必須、最大100文字 |
| テンプレート | 必須 |

## 8. API連携

- GET /api/templates
- GET /api/templates/{templateId}
- POST /api/projects/{projectId}/flows/from-template

## 9. DB連携

- TEMPLATE
- TEMPLATE_LANE
- TEMPLATE_STAGE
- TEMPLATE_NODE
- TEMPLATE_LINK
- FLOW
- FLOW_VERSION

## 10. エラー表示

- テンプレート一覧の取得に失敗しました
- テンプレート詳細の取得に失敗しました
- テンプレートからFlowを作成できませんでした

## 11. テスト観点

- テンプレート一覧が表示されること
- テンプレート詳細が表示されること
- Flow名未入力で作成できないこと
- テンプレートからFlowを作成できること
- 作成後にフローエディタへ遷移できること

## 12. 完了条件

本画面は、テンプレートから構造化データを持つFlowを作成できる仕様が定義された場合に完了とする。
