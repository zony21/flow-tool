## 3. 中核API

最重要APIはFlowVersion APIである。
フローエディタはFlowVersionDetailResponseを取得し、Lane、Stage、Node、Link、Commentを構造化データとして編集・保存する。

## 4. 画面との対応

- プロジェクト一覧: Project API
- フロー一覧: Flow API
- フローエディタ: FlowVersion API
- テンプレート選択: Template API
- 出力画面: Export API
- ログイン画面: Auth API

## 5. DBとの対応

- PROJECT: Project API
- FLOW: Flow API
- FLOW_VERSION: FlowVersion API
- LANE/STAGE/NODE/LINK/COMMENT: FlowVersion API
- TEMPLATE: Template API
- EXPORT_HISTORY: Export API
- USER/PROJECT_MEMBER: Auth/Permission API
