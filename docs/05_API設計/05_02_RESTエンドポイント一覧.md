# 05_02_RESTエンドポイント一覧

## 1. 目的

AI Flow Designer のREST API一覧を定義する。
画面、DB、DTO、テスト仕様へ展開できるよう、主要エンドポイントを整理する。

## 2. Auth API

| Method | Path | 用途 |
| --- | --- | --- |
| GET | /api/auth/github/login | GitHubログイン開始 |
| GET | /api/auth/github/callback | GitHubログイン完了 |
| GET | /api/auth/me | ログインユーザー取得 |
| POST | /api/auth/logout | ログアウト |

## 3. Project API

| Method | Path | 用途 |
| --- | --- | --- |
| GET | /api/projects | Project一覧取得 |
| POST | /api/projects | Project作成 |
| GET | /api/projects/{projectId} | Project詳細取得 |
| PUT | /api/projects/{projectId} | Project更新 |
| DELETE | /api/projects/{projectId} | Project削除 |
| GET | /api/projects/{projectId}/members | Projectメンバー取得 |

## 4. Flow API

| Method | Path | 用途 |
| --- | --- | --- |
| GET | /api/projects/{projectId}/flows | Flow一覧取得 |
| POST | /api/projects/{projectId}/flows | Flow作成 |
| GET | /api/projects/{projectId}/flows/{flowId} | Flow詳細取得 |
| PUT | /api/projects/{projectId}/flows/{flowId} | Flow更新 |
| DELETE | /api/projects/{projectId}/flows/{flowId} | Flow削除 |
| POST | /api/projects/{projectId}/flows/{flowId}/duplicate | Flow複製 |

## 5. FlowVersion API

| Method | Path | 用途 |
| --- | --- | --- |
| GET | /api/projects/{projectId}/flows/{flowId}/versions | Version一覧取得 |
| POST | /api/projects/{projectId}/flows/{flowId}/versions | Version作成 |
| GET | /api/projects/{projectId}/flows/{flowId}/versions/latest | 最新Version取得 |
| GET | /api/flow-versions/{flowVersionId} | Version詳細取得 |
| PUT | /api/flow-versions/{flowVersionId} | Version構造保存 |
| POST | /api/flow-versions/{flowVersionId}/duplicate | Version複製 |

## 6. Template API

| Method | Path | 用途 |
| --- | --- | --- |
| GET | /api/templates | Template一覧取得 |
| GET | /api/templates/{templateId} | Template詳細取得 |
| POST | /api/projects/{projectId}/flows/from-template | TemplateからFlow作成 |

## 7. Export API

| Method | Path | 用途 |
| --- | --- | --- |
| POST | /api/flow-versions/{flowVersionId}/export/mermaid | Mermaid生成 |
| POST | /api/flow-versions/{flowVersionId}/export/pdf | PDF生成 |
| POST | /api/flow-versions/{flowVersionId}/export/json | JSON生成 |
| GET | /api/flow-versions/{flowVersionId}/exports | 出力履歴取得 |

## 8. UserSetting API

| Method | Path | 用途 |
| --- | --- | --- |
| GET | /api/users/me/settings | 自分の設定取得 |
| PUT | /api/users/me/settings | 自分の設定更新 |
| POST | /api/users/me/settings/reset | 設定初期化 |

## 9. EditorLock API

| Method | Path | 用途 |
| --- | --- | --- |
| GET | /api/flow-versions/{flowVersionId}/lock | Lock状態取得 |
| POST | /api/flow-versions/{flowVersionId}/lock | Lock取得 |
| DELETE | /api/flow-versions/{flowVersionId}/lock | Lock解除 |

## 10. テスト観点

- 画面設計の主要操作に対応するAPIが存在すること
- FlowVersion保存APIが構造全体を扱えること
- Template適用APIがID再採番を前提としていること
- Export APIがVersion単位であること

## 11. 完了条件

主要画面と主要DBに対応するAPI一覧が定義されていること。
