# 07_08_Comment変換仕様

## 1. 本書の目的

CommentをMermaidへ変換する仕様を定義する。
Commentは通常コメント、Node紐付けコメント、AI専用メモを持つ。

## 2. 通常コメント

通常コメントはMermaidコメントとして出力する。

```mermaid
%% RFIDリーダ機種依存あり
```

## 3. Node紐付けコメント

Nodeに紐付くコメントは対象Nodeの直前または直後に出力する。

## 4. AI専用メモ

AI専用メモは出力オプションで含めるかを制御する。
標準では人向けMermaidには出力しない。

## 5. 独立コメント

独立コメントはflowchartでは補足ノードまたはMermaidコメントとして出力する。
初期実装ではMermaidコメントを標準とする。

## 6. エスケープ

コメント本文にMermaid構文へ影響する文字がある場合はエスケープする。

## 7. テスト観点

- 通常コメントを出力できること
- Nodeコメントが対象Node付近に出力されること
- AI専用メモの出力有無を切り替えられること

## 8. 完了条件

Commentの補足情報が用途に応じてMermaidへ反映されること。
