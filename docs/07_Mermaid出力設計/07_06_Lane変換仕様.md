# 07_06_Lane変換仕様

## 1. 本書の目的

LaneをMermaidへ変換する仕様を定義する。
Laneは担当機器、システム、作業者を表し、処理責務を明確にするために使用する。

## 2. flowchartでの扱い

flowchartではLaneをsubgraphとして表現する。
Nodeは所属Laneのsubgraph内に配置する。

## 3. sequenceDiagramでの扱い

sequenceDiagramではLaneをparticipantとして表現する。

## 4. 識別子

Lane IDを元にMermaid用の安全な識別子を生成する。
表示名にはlaneNameを使用する。

## 5. 表示順

LaneのdisplayOrderに従って出力する。

## 6. コメント

Laneのdescriptionは出力オプションによりコメントとして含める。

## 7. テスト観点

- Laneがsubgraphになること
- Laneがparticipantになること
- 表示順が維持されること

## 8. 完了条件

Laneが担当責務としてMermaidに反映されること。
