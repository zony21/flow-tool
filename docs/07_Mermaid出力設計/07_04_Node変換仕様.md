# 07_04_Node変換仕様

## 1. 本書の目的

NodeをMermaid記法へ変換する仕様を定義する。
Nodeは処理意味を持つため、NodeTypeに応じた記法へ変換する。

## 2. 入力項目

- nodeId
- nodeType
- displayName
- description
- laneId
- stageId

## 3. 識別子

Mermaid識別子にはnodeIdを使用する。
表示名変更でMermaid参照が壊れないようにする。

## 4. NodeType変換

| NodeType | Mermaid |
| --- | --- |
| START | `N001([開始])` |
| END | `N999([終了])` |
| PROCESS | `N001[処理]` |
| DECISION | `N002{判定}` |
| HEXAGON | `N003{{六角形}}` |
| BALLOON | `N004[吹き出し]` |
| IMAGE | `N005[画像]` |

## 5. ラベル

ラベルにはdisplayNameを使用する。
説明は出力オプションによりコメントとして含める。

## 6. エスケープ

Mermaid構文に影響する文字はエスケープまたは置換する。
改行は`<br/>`へ変換する。

## 7. テスト観点

- NodeTypeごとに正しい記法へ変換されること
- nodeIdが識別子に使用されること
- 表示名に特殊文字があっても壊れないこと

## 8. 完了条件

全初期NodeTypeをMermaidへ変換できること。
