# 06_12_ContextMenu設計

## 1. 本書の目的

ContextMenuの設計仕様を定義する。
右クリックまたは対象選択時の補助操作として、編集効率を高める。

## 2. 対象

- Canvas
- Node
- Link
- Comment
- Lane
- Stage

## 3. Canvasメニュー

- Node追加
- Comment追加
- 貼り付け
- 表示リセット

## 4. Nodeメニュー

- 編集
- 複製
- 削除
- Link作成開始
- Comment追加
- 詳細表示

## 5. Linkメニュー

- 編集
- 削除
- 条件編集

## 6. 権限制御

参照のみ権限では編集系メニューを非表示または非活性にする。

## 7. テスト観点

- 対象ごとにメニューが切り替わること
- 権限に応じて操作可否が変わること
- 削除時に確認が表示されること

## 8. 完了条件

ContextMenuから主要編集操作を安全に実行できること。
