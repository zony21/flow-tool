# 06_13_Shortcut設計

## 1. 本書の目的

Shortcutの設計仕様を定義する。
頻繁に行う編集操作をキーボードで実行できるようにし、作業効率を高める。

## 2. 基本ショートカット

| 操作 | キー |
| --- | --- |
| 保存 | Ctrl + S |
| Undo | Ctrl + Z |
| Redo | Ctrl + Y |
| コピー | Ctrl + C |
| 貼り付け | Ctrl + V |
| 削除 | Delete |
| 全選択 | Ctrl + A |
| Zoom In | Ctrl + + |
| Zoom Out | Ctrl + - |

## 3. 入力中の扱い

TextBox、Textarea、Dropdown入力中は、文字編集の標準ショートカットを優先する。

## 4. 権限制御

参照のみ権限では編集系ショートカットを無効にする。

## 5. エラー処理

保存失敗、削除不可、権限不足はToastで通知する。

## 6. テスト観点

- Ctrl + Sで保存が実行されること
- 入力中に文字編集を妨げないこと
- 参照権限で編集ショートカットが動かないこと

## 7. 完了条件

主要編集操作を安全にキーボードで実行できること。
