# Frontend

AI Flow Designer Frontend。

## 技術

- Vue 3
- TypeScript
- Vite
- Pinia
- Vue Router
- PrimeVue
- Vue Flow

## セットアップ

1. apps/frontendへ移動
2. 依存関係をインストール
3. 開発サーバーを起動

```bash
cd apps/frontend
npm install
npm run dev
```

## 主要ディレクトリ

```text
src/
	app/
	pages/
	layouts/
	features/
	components/
	stores/
	api/
	adapters/
	types/
	utils/
	router/
	styles/
```

## 環境変数

- VITE_API_BASE_URL
	- 既定値: http://localhost:5000
